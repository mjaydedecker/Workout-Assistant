# Workout-Assistant Project
Workout-Assistant is an Android application that tracks your workouts.

## Main Function
I want to use this Android application to track my strength training exercises.  To track my exercises I would like it to do the following:

- Be able to define a list of exercises and how may sets are normally performed for the excercise.
- Be able to allocate an exercise to a workout day.
- Be able to select a workout day to perform the exercises for a workout session.
- Be able to delete an exercise from a workout session.
- Be able to mark when an exercise set has been performed.
- Show when all the sets for an exercise have been completed.
- Be able to end a workout session and when it ends have the ability to add a note to the session for additional information.  The total duration of the workout session should also be recorded.
- Be able to type in the weight used for a the first exercise set of a workout session when a set has been completed and have that be remembered as the default for subsequent workout sessions.
- Be able to decrement a performed exercise just in case it was entered as performed in error.
- Be able to edit the weight used for an exercise session in case it was entered in error.
- Be able to show a history of workout sessions showing the exercises performed, the number of sets, weights, session duration and any notes for the workout session.
- Be able to export the history to a CSV file.
- Be able to set a rest timer of a user defined length.  This rest timer will remind the user when to start the next set after the completion of the previous set was entered by the user.  The reminder will be in the form of a sound and/or vibration depending on settings in the application.
- Show the progress of the rest time on the screen.
- Be able to set an inactivity timer of a user defined length.  This inactivity timer will remind the user when to continue exercising after the completion of the previous set was entered by the user.  The reminder will be in the form of a sound and/or vibration depending on settings in the application.
- Rest and inactivity timers will reset when a exercise set is marked as have been performed.
- Provide a setting to be able to keep the screen on during a workout.  If the setting is on, once a workout has starting and until it has ended, the screen will not automatically turn off.
- Provide a setting for a dark and light mode for the application.
- Be able have a setting for the app to choose kgs or lbs for weights used.
- Be able to edit the name of the workout day
- Be able to pause and resume a exercise session incorporating the pause into the exercise session total duration.
- Use a dumbbell icon for the app.
