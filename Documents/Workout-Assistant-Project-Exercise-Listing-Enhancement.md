# Workout Assistant Project Exercise Listing Enhancement
This enhancement is to prepopulatee exercises that can be selected for inclusion into a workout.

## Exercise Listing Enhancement
The purpose of this enhancement is to have list of exercies to choose from instead of having to type them in.  To achieve that I would like the following:
- Be able to continue to type in your own exercises.
- Prepopulate a list of exercises taken from the exercises.csv file.
- The following information should be provided for each prepopulated exerciese - Exercise Name, Force, Equipment, Primary Muscles, Secondary Muscles, Description (Step-By-Step Instructions), Videos, Images
- Exercises should be grouped by the primary muscle impacted.  If there are multiple primary muscles impacted, the should be listed for all impacted primary muscles.
- Be able to view the detailed information on the excercise.
- Be able to play a video to see the exercised performed.
- Be able to add prepopulated and custom entered exercises into Workout Day.
- When an exercise is added to a workout day, the number of set to be performed is entered.  This is a change from before where the number of set was associated with the exercise.  Since we are prepopulating exercises and not including a set count, we will more the set association to the Workout Day.
- Due to the changes in the structure of data, for the initial implemenation of this feature, we will have the data migration requirement.  This requirement will be required for subsequent revisions.
