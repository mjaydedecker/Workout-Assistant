# Workout-Assistant Project - Manual Testing Finds #3
Findings from the first set of manual testing.

## Issues
The following are issues that are to be corrected:
- During an exercise session, currently information on the exercise it not currently available.  The ability to see information on the exercise needs to be available during the exercise session.
- Currently, once an exercise has been added to a workout day, you are not able to change the number of sets.  The ability to change the number of set for an exercise needs to be able available.
- During an exercise session, with sound enabled for resting and inactivity timers, no sound plays.  Vibration timers to work but sound timers do not.

## Enhancements
The following are enhancements that are to be added to the project:
- When looking at the history of exercise sessions, I would like the ability to show the weight history of an exercise over time on a graph when an exercise is pressed.
